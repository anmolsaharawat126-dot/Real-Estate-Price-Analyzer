# 🏡 SmartEstate AI — Premium Real Estate Portal

SmartEstate AI is a 10/10 startup-level real estate portal built with Streamlit. It replicates the premium dark-mode, glassmorphism UI/UX of Next.js + Tailwind web applications. The platform comes with advanced features like AI-powered locality intelligence reports, dynamic price valuation, interactive 360° room walkthroughs, Aadhaar/PAN KYC badge verification, spam-protected call-back requests, builder construction timelines, and a fully functional agent CRM dashboard.

## 🚀 Key Features

### 1. 🤖 AI & Finance Intelligence
- **AI Price Prediction**: Evaluates listed property prices against real-time micro-market estimates based on size, BHK, property type, and locality.
- **AI Locality Reports**: Generates comprehensive scores (out of 100) detailing safety, water/power cut history, broadband providers, flood risks, and upcoming infrastructure/metro projects.
- **Finance Suite**: Interactive Home Loan EMI planner, maximum loan eligibility calculator, and state-wise stamp duty calculators (with women-buyer rebates).
- **Gemini NLP Search**: (Optional) Fully integrated NLP consultant powered by Gemini API. Click-to-trigger suggestion chips are included.

### 2. 🛡️ Trust & Safety (Aadhaar KYC)
- **Mock OTP Verification**: Owners and agents can verify their profiles using Aadhaar and PAN inputs to get visual verification badges (`👑 Verified Owner` / `🎖️ Trusted Agent`).
- **Spam Protection**: Contact numbers of listings can be hidden. Buyers must click "Request Call Back," which agents/owners can approve or reject from their dashboard before sharing details.
- **Duplicate Listings Detection**: Automatically flags duplicate properties listed in the same micro-market with matching sizes.

### 3. 📅 Interactive Property Tours
- **360° Virtual Tour**: Seamlessly rotate and inspect rooms (Living Room, Master Bedroom, Kitchen, Balcony) with detailed ambient summaries.
- **Visit Scheduler**: Schedule physical site visits or virtual Zoom/WhatsApp tours at preferred dates and times.

### 4. 📊 Fully Featured Dashboards
- **Buyer Hub**: Tracks saved wishlist properties, scheduled visits, and pending/approved callback requests.
- **Agent CRM Portal**: Manage client pipelines (New Leads, Contacted, Site Visit, Negotiation, Closed) with deal-stage updates, follow-up reminders, and real-time commission metrics.
- **Admin Panel**: Queue systems for approving new listings before they go live on the portal.

---

## 🛠️ Technology Stack
- **Frontend & Layout**: Streamlit (Python) with highly customized CSS overrides (Outfit & Outfit fonts, custom scrollbars, glassmorphism panels, and fade-in animations).
- **Maps**: Folium & Streamlit-Folium (using premium Google Maps road/satellite layers).
- **Data Store**: Single-file JSON database structure in `/data` for lightweight, zero-config setup on Streamlit Cloud.
- **Calculations**: Math-mode loan eligibility, stamp duty rates, and micro-market appraisal hashes.
